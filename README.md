#  XYZ Pizza
A modern React pizza ordering app that lets users browse the menu, select pizzas, provide their address, and optionally choose a **priority delivery** for an additional fee.  
Includes loading & error handling for smooth UX.

Live Demo: **https://xyz-pizza.onrender.com**
